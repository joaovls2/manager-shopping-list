# 🛒 Gerenciador de Lista de Compras

Um projeto simples desenvolvido em **C#** para praticar os fundamentos da linguagem, manipulação de listas, estruturas de controle e leitura/escrita de arquivos (`System.IO`).

Os itens da lista são armazenados em memória durante a execução e podem ser salvos em um arquivo `.txt`, permitindo que sejam carregados novamente na próxima vez que o programa for executado.

---

## ✨ Funcionalidades

- ➕ Adicionar itens à lista de compras
- ❌ Remover itens existentes
- ✏️ Atualizar o nome de um item
- 📋 Listar todos os itens
- 🗑️ Limpar toda a lista
- 💾 Salvar a lista em um arquivo `.txt`
- 📂 Carregar automaticamente a lista salva ao iniciar o programa

---

## 🛠️ Tecnologias utilizadas

- C#
- .NET
- `System.IO`
- `List<T>`

---

## 📚 Conceitos praticados

Durante o desenvolvimento deste projeto foram praticados diversos conceitos fundamentais da linguagem C#:

- Variáveis e constantes
- Estruturas condicionais (`if` / `else`)
- Estrutura de decisão (`switch`)
- Estruturas de repetição (`while` e `for`)
- Coleções (`List<T>`)
- Manipulação de arquivos (`System.IO`)

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de consolidar os conhecimentos iniciais em **C#** através da implementação de um CRUD 
em aplicação de console, praticando manipulação de coleções e persistência de dados utilizando arquivos de texto.
